About This Project

Get top 250 popular books' data from douban reading utilizing Python web crawler. The data include book info link, book cover link, title, one-sentence pitch, author, press, publication data and price.

SQLite database is used to store the data. Echarts is used for data visualization to show the distribution of rating of books. And Word Cloud is a novelty visual representation of text data, typically used to depict keywords of one sentence pitch. This format is useful for quickly perceiving the most prominent terms to determine its relative prominence.

  • Python Crawler - Data Processing
  • SQLite - Data Storage
  • Flask - Werkzeug + Jinja Website Development
  • Echarts - Distribution of Rating Score
  • Word Cloud - Visual Representation of Text Data
